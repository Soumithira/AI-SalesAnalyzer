# AI Sales Analyzer

## Project Objective

The AI Sales Analyzer is a data analytics dashboard designed to analyze sales data and generate useful business insights.  
It helps users understand sales performance, track revenue trends, and forecast future sales using machine learning models.

The system provides an interactive dashboard built with Streamlit where users can upload datasets and view analytics reports.

Machine learning models for future sales prediction

## How to Run the Project

Step 1: Install Python libraries

pip install -r requirements.txt

Step 2: Run the Streamlit application

streamlit run app.py

Step 3: Open browser

http://localhost:8501
